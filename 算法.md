						爱因斯坦的难题 
用python/go/php三种语言进行依次实现

爱因斯坦曾出过这样一道有趣的数学题：有一个长阶梯，若每步上2阶，最后剩1阶；若每步上3阶，最后剩2阶；若每步上5阶，最后剩4阶；若每步上6阶，最后剩5阶；只有每步上7阶，最后刚好一阶也不剩。
第一种python
``` python
x = 7
i = 1
flag = 0
while i <= 100 :
    if (x % 2 == 1) and (x % 3 == 2) and (x % 5 == 4) and (x % 6 == 5) :
        flag = 1
        break
    else :
        x = 7 * (i+1)
    i = i + 1

if flag == 1 :
    print("数字为：", x)
else:
    print("不在100范围内")
```

第二种go
``` go
package main

import "fmt"

func main() {
	x := 7
	i := 1
	flag := 0
	for {
		if i <= 100 {
			if (x % 2 == 1) && (x % 3 == 2) && (x % 5 == 4) && (x % 6 == 5) {
				flag = 1
			} else {
				x = 7 * (i + 1)
			}
			i++
		} else {
				break
		}
	}
	if flag == 1 {
		fmt.Println("num:", x)
	} else {
		fmt.Println("该数不在该范围内")
	}
}
```

第三种 php
``` php

    if (($x % 2 == 1) && ($x % 3 == 2) && ($x % 5 == 4) && ($x % 6 == 5)) {

```
